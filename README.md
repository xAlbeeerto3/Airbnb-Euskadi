# English 
This project consists of an exploratory data analysis (EDA) for Airbnb in the autonomous community of Euskadi. The objective is to find patterns and trends in rental prices. To achieve this, the datasets extracted from the InsideAirbnb website have been preprocessed and cleaned to access characteristics of the properties, hosts, types of rooms, and prices according to the supply and demand in the provincial capitals Vizcaya, 츼lava, and Guip칰zcoa.

Additionally, a heat map has been created based on the price of localities, a map with the actual number of properties by area, and a Power BI dashboard to better represent the results obtained from the study. All of this is under one premise: *Is Euskadi a profitable option and does it provide tourists with more affordable accommodations to enjoy at any time of the year?*
